  • Explain the difference between accessing data from arrays and objects.

    • The difference with accessing data from arrays and objects,

    An array is able to hold more than one value at a time, under a single name, and you can access the values by referring to an index number using the square brackets.

    An object is similar to an Array can hold more than one value, is a real object like a car that has properties like colour, name and model. They can be accessed like this objectName.propertyName or objectName["propertyName"] using the square brackets.
